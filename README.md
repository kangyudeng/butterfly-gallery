# 蝴蝶画廊 — 本地静态站点

说明：这是为你提供的图片集合制作的静态画廊站点。入口处有 3D 粒子汇聚成蝴蝶的动画，点击蝴蝶可进入主页并查看按类别分组的图库。

快速预览（本地）：

1. 打开终端，进入该目录：

```bash
cd ~/Desktop/蝴蝶照片
```

2. 在该目录启动简单静态服务器（推荐使用 Python 3）：

```bash
python3 -m http.server 8000
```

3. 在浏览器打开：

```
http://localhost:8000/
```

注意：为了简洁，图片保持在当前目录中，`images.json` 使用的是这些文件名。若你移动图片到子目录，请更新 `images.json` 中的路径。

部署建议：可把整个文件夹上传到 Netlify / Vercel / GitHub Pages（将仓库根设置为本目录）。
